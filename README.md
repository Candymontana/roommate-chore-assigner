# Roommate Chore Assigner 🏠🐍 by Candymontana (@candy.py)

## Description
This is my very first Python project! I created this script to solve a common problem: deciding who does what at home. It's a simple tool that takes a list of roommates and a list of chores, then randomly pairs them up.

I am currently a student in **Marketing and Commerce** (Master I) and I'm learning Python via **freeCodeCamp** to eventually pivot into **Data Analysis**.

## My "Build in Public" Process 🛠️
I believe in being honest about how I code:

## Features
- Dynamic input for roommate names.
- Dynamic input for the list of chores.
- Randomized assignments using Python's `random` module.

## How to use
1. Clone this repository.
2. Run the script: `chore_assigner.py`
3. Follow the prompts to enter names and chores.
